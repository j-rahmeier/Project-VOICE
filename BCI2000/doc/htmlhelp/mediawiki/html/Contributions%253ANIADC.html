          Synopsis

          A data acquisition component for National Instruments AD boards.

          Location

          http://www.bci2000.org/svn/trunk/src/contrib/SignalSource/NIDAQ

          Versioning

          Author

          Gerwin Schalk, Wadsworth Center, NYSDOH.

          Version History

          V1.00 - 08/09/2002 - First start and also first functioning version

          Source Code Revisions

          • Initial development: 103
          • Tested under: 169
          • Known to compile under: 1476
          • Broken since: --

          Parameters

          BoardNumber

          The NI-ADC board's device number.

          States

          None.

          Comments

          This source module was tested using National Instruments' 6.9.x drivers. It may or may not work with the newer MX drivers.
