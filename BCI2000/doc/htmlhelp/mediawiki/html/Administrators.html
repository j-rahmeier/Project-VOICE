            BCI2000 Glossary

            Administrators

            Administrator Role

            Administrators are responsible for setup and maintenance of software and user accounts. Although new user accounts may be created by existing users, it may be necessary to contact an administrator to:

            • get more than the default set of user privileges,
            • disable/remove a user account,
            • reset a user's password.
